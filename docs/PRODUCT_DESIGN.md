# 🌾 Dehqonjon — Fermerlar uchun Marketplace + AI-yordamchi

## Общая концепция

**Миссия:** Простое, быстрое и понятное приложение для фермеров, объединяющее торговлю и умные консультации.

**Философия дизайна:**
- Mobile-first (80% пользователей — с телефона в поле)
- Минимум действий до результата
- Крупные элементы, высокий контраст
- Понятно без обучения

## 🔧 Рекомендации по UX для фермеров

### 1. Учёт условий использования
- **Яркое солнце** → Высокий контраст, тёмный текст на светлом фоне
- **Грязные руки** → Крупные кнопки (min 44px), большие отступы между элементами
- **Плохой интернет** → Оффлайн-режим, кэширование, сжатие изображений
- **Старые телефоны** → Лёгкий код, минимум анимаций

### 2. Упрощение ввода
- Голосовой ввод в поиске и чате
- Автозаполнение региона по GPS
- Шаблоны объявлений по категориям
- Быстрые ответы в чате

### 3. Доверие к ИИ
- Всегда показывать вероятности (%)
- Предупреждения: "Это не замена агроному"
- Возможность связаться с экспертом
- История консультаций

### 4. Доступность
- Размер шрифта: минимум 16px
- Контраст: минимум 4.5:1
- Поддержка увеличения текста
- Альтернативный текст для изображений

### 5. Быстрые действия
- Повторить прошлое объявление
- Быстрый звонок продавцу
- Поделиться товаром в WhatsApp
- Сохранить диагноз ИИ в PDF
